How to Use

For Kobo: Connect your device via USB and copy the EPUB files to the root directory.

For Kindle: Email the EPUB to your Kindle email address or use Calibre to convert to MOBI.

For other e-readers: Most modern e-readers support EPUB format directly.

How These EPUBs Are Generated

All EPUBs are automatically generated from the Sefaria API using Python scripts that:

  • Fetch Hebrew texts and translations from Sefaria's open database
  • Support multiple English translations (JPS, Koren, Community, etc.)
  • Include classical commentaries like Rashi, Ibn Ezra, and Ramban
  • Optimize layouts for e-reader screens (especially Kobo Color)
  • Handle Hebrew RTL text and proper font embedding
  • Generate responsive layouts that adapt to portrait/landscape

GitHub Actions automatically builds fresh EPUBs whenever the code is updated.
You can also fork the repository and customize the generation for your preferences.
